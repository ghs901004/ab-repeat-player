About the tool

Practice short audio loops with a local MP3 file

AB Repeat Player helps language learners, musicians, transcription workers, and anyone who studies spoken audio repeat one small part of an MP3 file until it becomes easier to hear and understand.

Choose a language version

The same player is available in Chinese, English, Spanish, and Japanese. Each version has its own interface text, help pages, privacy information, and contact page.

Audio stays in the browser

You select an MP3 from your own device. The file is played locally by your browser and is not uploaded to this website, which makes it suitable for private study materials.

Use short A/B sections

A good practice loop is usually ten to twenty seconds long. Set point A, set point B, repeat the section slowly, then return to normal speed when the sound becomes familiar.

Read practical guides

Before practicing, you can read the English practice guide and FAQ, or open the matching guide from any language version.
