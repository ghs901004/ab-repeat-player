音频文件

AB 复读机允许你从设备中选择 MP3 文件。音频在浏览器本地处理，不会上传到我们的服务器。

我们收集的信息

本站不提供账号、登录或文件存储功能。播放器不会主动收集你的姓名、邮箱或上传音频文件。

托管和技术数据

本站托管在 GitHub Pages 上。GitHub 或相关基础设施提供商可能会为了安全、运营和防滥用目的处理标准技术数据，例如 IP 地址、浏览器类型、请求时间和访问页面。

广告和 Cookie

我们可能使用 Google AdSense 等第三方广告服务。这些服务可能会根据其政策使用 Cookie、网页信标、设备标识符和 IP 地址信息来投放、衡量和个性化广告。

联系

如有隐私相关问题，请使用联系页面，或在公开 GitHub 仓库中提交 issue。
